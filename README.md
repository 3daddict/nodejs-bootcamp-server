# DevCamper API

> Backend API for DevCamper application, which is a bootcamp directory website.

## Usage
Rename "config/example_env" to "config/config.env" and update the values.

## Install Dependancies
```
yarn install
```

## Run App
```
yarn dev
```

## Run in Production Mode
```
yarn start
```
